## 什么是WyYBE

WyY Blog Editor是一款静态Html生成工具，它可以依据既定规则自动生成静态Html文件，并自动部署至Git托管平台。你可以使用它来创建博客，编写文档等等。

## 如何工作

WyYBE将按照既定规则渲染文档内的指定内容（包括布局，列表渲染，变量渲染）。同时你还能够使用预先编写好的JavaScript文件实现站内搜索等功能。

## 拓展
